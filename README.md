# MPPT Buck Converter using Arduino and MATLAB Simulink

This project presents the design and simulation of a Buck Converter with Perturb and Observe (P&O) Maximum Power Point Tracking (MPPT) for a Solar PV system.

The system is designed to charge a 12V battery using a PV array. 
PWM signal is generated using Arduino, and the converter is simulated in MATLAB Simulink.

## Features

- PV Array modeling
- Buck converter design
- Perturb and Observe MPPT algorithm
- Arduino PWM control
- Low ripple converter design
- 10 kHz switching frequency

## Parameters

- Input voltage (PV) : 18V
- Output voltage : 12V
- Battery : 12V 7Ah
- Switching frequency : 10 kHz
- Inductor : 680 uH / 1 mH
- Output capacitor : 470 uF
- Input capacitor : 1000 uF

## Software Used

- MATLAB Simulink
- Arduino IDE
